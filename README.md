# automatic-firefighter-robot
An autonomous robotic system designed to detect and extinguish fire using flame sensors, embedded control, and real-time navigation, enhancing safety in hazardous environments.

🔥 Automatic Firefighter Robot
An intelligent, autonomous robotic system designed to detect and extinguish fire hazards in real-time. This project integrates embedded systems, sensor technology, and control algorithms to enhance safety in environments prone to fire risks.


📌 Overview
The Automatic Firefighter Robot is built to identify fire sources using flame sensors and respond immediately by navigating toward the hazard and activating an extinguishing mechanism. The system reduces human risk by handling fire incidents in early stages, especially in hazardous or inaccessiblnavigation

🚀 Features
🔍 Fire Detection: Uses flame sensors to detect fire presence and direction
🤖 Autonomous Navigation: Moves toward the fire source using motor control logic
💧 Fire Suppression System: Activates water pump or extinguisher automatically
🧠 Microcontroller-Based Control: Efficient processing using embedded system
⚡ Real-Time Response: Immediate detection and action
🛑 Obstacle Avoidance (Optional): Ultrasonic sensor-based navigaSystem

🛠️ Technologies Used
Embedded C / Arduino Programming
Microcontroller (e.g., Arduino Uno)
Flame Sensors
Motor Driver (L298N / L293D)
DC Motors
Water Pump / Fan Module
Ultrasonic Sensor (HC-SR04) (optional)
Power Supply System

⚙️ Working Principle
Flame sensors continuously monitor the surroundings.
When fire is detected, the robot identifies its direction.
The controller processes sensor data and drives motors accordingly.
The robot moves toward the fire source.
Upon reaching a safe distance, the extinguishing system is activated.
